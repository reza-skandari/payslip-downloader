Windows may warn you on first launch

Because this app isn't published through the Microsoft Store, Windows Defender SmartScreen may show "Windows protected your PC". This is normal for independently distributed software. To proceed: click More info, then Run anyway. The app is open source — you can review the code on GitHub.

How it works

Step 1

Run the app

Double-click the exe. First launch downloads the browser it needs (~1 min). Every run after is instant. If Windows warns "unrecognised app", click More info → Run anyway.

Step 2

Log in with your Imperial account

A browser window opens for Microsoft login and MFA. It closes automatically once you're in.

Step 3

Choose a range and download

Pick start and end month, choose a folder, and click Download. Each payslip takes a few seconds — be patient.

  • Windows 10 or 11
  • Imperial Microsoft 365 account
  • Internet connection
